          开发语言：JAVA
   ─────────────────────────────────
说明：
本demo仅供参考，用户可根据自己的需要自己做实现；

※注意※
需要配置的文件是：
service/CommonService.java 根据自己的业务逻辑对订阅消息做处理
handler/SendDataThread.java 根据自己的业务逻辑取数据进行xml封装
conf/cfg.properties
其中需修改的字段说明：
	--db_Driver 	数据库驱动（根据业务需要修改自己的数据库路径）
	--db_Connection 数据库地址
	--db_UserName	数据库用户名
	--db_Password	数据库密码
	--AppId			项目id，防止同一项目重复启动
	--sourceID		订阅服务时注册的业务id
	--ReadTopic		项目监听的主题名称m
	--WriteTopic	项目发送给集成平台数据的主题名称
	--ProcedureType 程序启动类型（发布/订阅功能）

本代码示例（demo）中模拟获取订阅消息
上层消息的获取方式集成至lib/kafkatools.jar包中；
获取到订阅数据以后业务处理请根据自己业务需要做处理；
模拟发布数据
所要发布的数据请根据自己的业务逻辑进行抽取并封装
